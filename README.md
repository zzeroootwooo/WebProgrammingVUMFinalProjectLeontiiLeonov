# Paradise Hotel Project

Full-stack hotel reservation project with:

- `backend/` - Express + SQLite API
- `frontend/` - React + Vite client

## Quick Start

### 1. Run backend

```bash
cd backend
npm install
npm run dev
```

Backend URL:

```text
http://localhost:4000
```

### 2. Run frontend

Open a second terminal:

```bash
cd frontend
npm install
npm run dev
```

Frontend URL:

```text
http://localhost:5173
```

## Frontend Environment

If needed, create `frontend/.env`:

```bash
VITE_API_BASE_URL=http://localhost:4000
```
